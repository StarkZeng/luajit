
this is a simple xCode project to tell how to use luajit

1.any help document can be find in  http://luajit.org/
2.to crate luajit command line tools : make & make install
you will find the tools and lib in src 

3.to use luajit in xcode or android project ,add .a/.so file to project 
and create .out file form lua file use "luajit -bg" ,see  running in http://luajit.org/
